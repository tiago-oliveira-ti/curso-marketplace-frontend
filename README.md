# Marketplace de Cursos Online

Projeto frontend desenvolvido em **HTML, CSS e JavaScript puro**, simulando uma plataforma de cursos online.

## Funcionalidades
- Listagem de cursos com imagem, título e preço
- Filtro por categoria com indicação visual do filtro ativo
- Favoritar cursos com feedback visual (salvo no localStorage)
- Carrinho de compras com cálculo automático e remoção de itens
- Contador de itens no botão do carrinho
- Persistência do carrinho no localStorage
- Login fake (usuário: admin, senha: 123)
- Dark mode com persistência no localStorage
- Notificações via toast (sem alert())
- Fechar modais clicando fora (overlay)

## Tecnologias
- HTML5
- CSS3 (Flexbox e Grid)
- JavaScript (DOM, eventos, localStorage)

## Responsividade
- Layout adaptado para **mobile e desktop**
